Semplicità per i cittadini nella fruizione dei servizi comunali

Le informazioni esposte in questa piattaforma sono tratte dalle carte dei servizi comunali, dalla sezione modulistica del sito web istituzionale, dai link degli applicativi presenti nel portale dei servizi online, e inserite in questo file di Google spreadsheet che alimenta automaticamente questa visualizzazione.
Lo scopo di questa piattaforma è consentire all'utente una facile consultazione delle informazioni per fruire i servizi comunali, facendo restare l'utente nella stessa pagina web senza dargli l'onere di trovare le informazioni sui servizi in diverse sezioni del sito istituzionale.

Strumenti

Gli strumenti principali usati per realizzare questa visualizzazione sono quattro:

  • Simile Exhibit, per realizzare l'interfaccia di navigazione del catalogo;
  • Twitter Boostrap, per dare un po' di stile alle pagine pubblicate;
  • GDrive Sheet, per raccogliere e distribuire i dati che fanno da sorgente;
  • GitHub Pages, per l'hosting dei file che costituiscono questa visualizzazione.
